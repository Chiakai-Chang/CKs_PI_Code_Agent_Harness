# Graphify Bridge

This extension bridges [safishamsi/graphify](https://github.com/safishamsi/graphify) into the Pi Harness.

## Key Concepts
*   **AST Analysis**: tree-sitter parses over 20 programming languages to extract class, function, and import relationships.
*   **71.5x Token Efficiency**: AI queries the local knowledge graph instead of traversing massive raw codebases.
*   **Obsidian/Mermaid Viz**: Exports directly to SVG, HTML, Obsidian vault, and Neo4j cypher files.
